# AI Studio Chat Lite

UI chat đơn giản cho AI Studio client.

## Chạy

```bash
node UI_lite/index.js
```

## Tính năng

- Chat với Gemini qua AI Studio
- Streaming response realtime
- Commands đơn giản: `exit`, `clear`
- Đơn giản, chỉ 1 file duy nhất

## So sánh với UI/

### UI_lite (Đơn giản)
- 1 file duy nhất
- Không chia component
- Dễ hiểu, dễ sửa
- Phù hợp cho demo nhanh

### UI/ (Chuyên nghiệp)
- Nhiều components
- Hooks tách riêng
- Status bar, logo, animations
- Phù hợp cho production
